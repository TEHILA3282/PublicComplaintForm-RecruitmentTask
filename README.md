## שיטת הבנייה

הפרויקט בנוי בהפרדת אחריות בין שכבות.

### Angular

- Components אחראים על התצוגה.
- Services אחראים על הלוגיקה המשותפת וקריאות HTTP.
- Reactive Forms אחראים על ניהול נתוני הטפסים והוולידציות.

### ASP.NET Core

- Minimal API אחראי על ה־Endpoints.
- הלוגיקה העסקית נמצאת בשירותים ייעודיים.
- Dapper אחראי על הגישה למסד הנתונים.
- המודלים מגדירים את מבנה הנתונים המועברים בין הלקוח לשרת.

השירותים מוזרקים באמצעות Dependency Injection של ASP.NET Core, דבר המאפשר הפרדת אחריות, תחזוקה נוחה והרחבה עתידית של המערכת.

במסגרת המשימה הקפדתי להשתלב במבנה הקיים של הפרויקט ולא להוסיף ארכיטקטורה חדשה, כדי לשמור על אחידות הקוד וקלות התחזוקה.
